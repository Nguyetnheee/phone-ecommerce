# DESIGN SKILL — ECOMMERCE UI

Design direction:
- Clean ecommerce interface
- Mobile-first
- Modern but simple
- Student-project friendly
- Prioritize clarity over animation

Main pages:
- Home
- Product List
- Product Detail
- Cart
- Checkout
- Login/Register
- Order Tracking
- Policy Pages
- Admin Dashboard

UI rules:
- Use consistent spacing.
- Use consistent button styles.
- Product card must show:
  - product image
  - product name
  - price
  - storage/RAM if available
  - add to cart button
- Header must include:
  - logo
  - search
  - navigation
  - cart icon/link
  - login/account link
- Footer must include:
  - contact info
  - policies
  - ecommerce legal notice placeholder

Do not:
- Use too many colors.
- Use complex animations.
- Create inconsistent layouts across pages.
